  • Git 워크트리 기반 병렬 에이전트 관리: 오르카는 Git 워크트리 개념을 활용하여 코덱스, 헤르메스 등 다양한 AI 에이전트들을 독립적인 작업 공간에서 병렬로 실행 및 관리함으로써, 복잡한 개발 프로젝트의 효율성을 크게 향상시킵니다.

  • 직관적인 웹 UI 상호작용 및 GitHub 통합: 웹 화면을 터미널처럼 띄워 특정 UI 요소를 에이전트에게 직접 지시하거나 주석을 달 수 있으며, GitHub의 PR, 이슈, 머지 등 핵심 개발 워크플로우를 오르카 내에서 통합 관리하여 개발 생산성을 극대화합니다.
